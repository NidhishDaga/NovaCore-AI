# 🧠 NovaCore AI

🌐 **Live Demo:** [Try NovaCore AI](https://novacore-ai-nidhishdaga.streamlit.app))
💻 **GitHub:** [View Source Code](https://github.com/NidhishDaga/NovaCore-AI))

---

A premium multimodal AI assistant platform 
built with Python and Streamlit.

## ✨ Features
- 🧠 Multi-turn conversation memory
- 📄 Document Q&A (PDF & TXT)
- 👁️ Vision AI (image analysis)
- 🎤 Voice input
- 🔐 Google authentication
- 💾 Private chat per user (SQLite)
- 🌐 Live and accessible worldwide

## 🛠️ Tech Stack
- Python & Streamlit
- Groq API (LLM inference)
- Google OAuth
- SQLite
- Pillow & PyPDF2

## 🤖 Supported Models
- LLaMA3
- DeepSeek R1
- Mistral
- Gemma
- LLaVA Vision
